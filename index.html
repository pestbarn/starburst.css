Starbursts in CSS?!

A mixin for doing it the easy way

.rectangular{} 

.burst_rect_1 {
    @include starburst(top,12,chocolate,darkorange,1);
    height: 200px; width: 200px;
}
.burst_rect_2 {
    @include starburst(both,11,darkorange,chocolate,1);
    height: 200px; width: 200px;
}
.burst_rect_3 {
    @include starburst(bottom,12,darkorange,chocolate,1);
    height: 200px; width: 200px;
}

.circular{} 

.burst_circ_5 {
    @include starburst(top,12,chocolate,darkorange,1,1);
    height: 200px; width: 200px;
}
.burst_circ_6 {
    @include starburst(both,11,darkorange,chocolate,1,1);
    height: 200px; width: 200px;
}
.burst_circ_7 {
    @include starburst(bottom,12,darkorange,chocolate,1,1);
    height: 200px; width: 200px;
}

// Anything that can have a background will do just fine 

.burst_rect_4.skew {
    @include starburst(both, 6, chocolate, darkorange, 1, 0);
    @include transform(skew(20deg));
    height: 200px; width: 200px;
}
.burst_circ_8.egg {
    @include starburst();
    width: 126px;
    height: 180px;
    @include border-radius(50% 50% 50% 50% / 60% 60% 40% 40%);
}

Entire sections 

section {
    @include starburst(bottom,9,darkred,maroon,-1);
    height: 50vh; width: 100%;
}

Neat, huh?

The only occasion when you shouldn't use Starburst is when you need to use the :before and :after pseudo-elements for something else. Starburst needs these to work properly.
