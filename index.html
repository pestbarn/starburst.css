Starbursts in CSS?!

A mixin for doing it the easy way

Rectangular 

.burst_rect_1 {
        @include starburst(top,12,chocolate,darkorange,1);
        height: 200px; width: 200px;
}
.burst_rect_2 {
        @include starburst(both,11,darkorange,chocolate,1);
        height: 200px; width: 200px;
}
.burst_rect_3 {
        @include starburst(bottom,12,darkorange,chocolate,1);
        height: 200px; width: 200px;
}

Circular 

.burst_circ_1 {
        @include starburst(top,12,chocolate,darkorange,1,1);
        height: 200px; width: 200px;
}
.burst_circ_2 {
        @include starburst(both,11,darkorange,chocolate,1,1);
        height: 200px; width: 200px;
}
.burst_circ_3 {
        @include starburst(bottom,12,darkorange,chocolate,1,1);
        height: 200px; width: 200px;
}

Entire sections (with content inside!) 

section {
        @include starburst(bottom,9,darkred,maroon,-1);
        height: 50vh;
}
